# ReportingTask
#Instructions
- Clone master branch
- Open "ReportingTask" solution
- You would need to install two packages from nuget:
    "EPPlusFree" Version="4.5.3.8"
    "Newtonsoft.Json" Version="13.0.3"
- Run ReportingTask project.
